Getting started

1. Install a recent JDK (Java 17+ recommended).

2. Clone the repository:

git clone https://github.com/dwavy079/ChessGame.git
cd ChessGame

3. Compile and run:

javac *.java
java ChessFrontend
Game features
  • Full chess rules: legal moves, check, checkmate, stalemate.
  • Play as White vs a simple random-move bot, or two-player mode.
  • Highlighted legal moves and capture squares for the selected piece.
  • Captured pieces panel and scrolling move history.
  • Pawn promotion dialog (Queen, Rook, Bishop, Knight) for human moves.
Under the hood

The engine is built around a Board class that tracks an 8×8 array of ChessPiece objects. Each piece type implements its own isValidMove logic, while shared path-checking lives in the base class. The UI layer (ChessFrontend) is a pure Swing application with no external dependencies.
